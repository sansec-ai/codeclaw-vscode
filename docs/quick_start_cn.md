# 快速开始指南

通过微信在你的VSCode项目内执行 Claude Code

## 前置条件

| 要求 | 说明 |
|------|------|
| **VSCode** | >= 1.85.0 |
| **微信** | iOS 微信，支持 ClawBot 功能 |

> **运行时无需 Node.js** — Claude Code CLI 已内置打包在插件中。
> **已经在使用 Claude Code？** 直接跳到第三步 — Code Claw 会自动读取你已有的配置。

## 第一步：安装插件

### 方式一：从 VSIX 安装（推荐）

```bash
code --install-extension codeclaw-vscode-0.1.77.vsix
```

或在 VSCode 中：`Ctrl+Shift+P` → `Extensions: Install from VSIX...` → 选择文件。

### 方式二：从 VSCode Marketplace 安装

打开扩展视图（`Ctrl+Shift+X`），搜索 **"Code Claw"**，点击 **Install**。

## 第二步：打开项目

在 VSCode 中打开一个项目文件夹，这就是 Claude 的工作目录。

```
文件 → 打开文件夹 → /path/to/your/project
```

## 第三步：连接微信

1. 点击 VSCode 左侧 Activity Bar 的 **Code Claw 图标** (![](../icon.svg))
2. 在面板中点击 **"连接微信"** 按钮
3. 弹出二维码 — 用 **手机微信** 扫描
4. 等待 "✅ 绑定成功" 提示

底部状态栏应显示 `WeChat: ✅ 已连接`。

## 第四步：开始对话

打开手机微信，找到 **ClawBot** 会话，发送消息：

```
你好，请介绍一下当前项目的结构
```

Claude Code 会分析你的项目并直接在微信中回复。

## 常用命令

| 命令 | 功能 |
|------|------|
| `/help` | 显示所有可用命令 |
| `/new` | 开启新会话（清除上下文） |
| `/model claude-sonnet-4-6` | 切换 Claude 模型 |
| `/mode 2` | 切换到自动接受文件编辑模式 |
| `/mode 3` | 跳过所有权限检查 |
| `/status` | 查看当前会话信息 |

**权限模式快捷键：** `0` = 仅规划，`1` = 默认逐次确认，`2` = 自动接受编辑，`3+` = 跳过所有权限

## 使用技巧

- **长回复**：Claude 的完整回复会作为一条消息发送（不拆分）。较大的输出可能需要稍等片刻。
- **流式输出**：默认开启，Claude 的中间工具调用记录在 VSCode 输出通道中（`查看 → 输出 → Code Claw`），只有最终结果发送到微信。
- *todolist 追踪**：Claude 使用 TodoWrite 时，任务进度会自动推送到微信，显示可视化的进度条。
- **权限拒绝**：如果 Claude 说完成了但实际工具被拒绝，你会看到 `/mode` 切换提示。用 `/mode 3` 跳过所有权限。
- **自动重连**：VSCode 重启后，如果打开了工作区，插件会自动使用已保存的凭证重新连接。
- **断开与重连**：点击"断开连接"停止 daemon，账号数据会保留。点击"重新连接"即可立即恢复，无需重新扫码。
- **多项目支持**：你的微信账号可以在任何 VSCode 项目中使用。切换项目后只需点击"重新连接"，绑定会自动更新。
- **会话过期**：如果微信会话过期，点击侧边栏的 "重新绑定" 扫描新的二维码即可。

## 常见问题

### Q: 微信 ClawBot 只能绑定一个 VSCode 项目，如果要换项目怎么办？

A: 不需要解绑！你的微信账号可以在任何 VSCode 项目中使用。切换到新项目后：

1. 在新项目的 VSCode 中点击左侧 Activity Bar 的 **CodeClaw图标**
2. 点击 **"重新连接"** 按钮
3. 插件会自动使用已保存的凭证连接，无需重新扫码

如果旧项目的 VSCode 窗口还在运行，需要先在那个窗口中点击"断开连接"（同一账号同一时间只能在一个 VSCode 窗口中活跃）。

### Q: "断开连接"和"重新绑定"有什么区别？

A: 
- **断开连接**：停止 daemon，但保留账号数据。下次点"重新连接"可以立即恢复，无需扫码。
- **重新绑定**：断开连接并清除账号数据，需要重新扫描二维码绑定新的微信账号。

### Q: 已经在使用 Claude Code，需要额外配置吗？

A: 不需要。如果你已安装 Claude Code 插件并配置了 `claudeCode.environmentVariables`，Code Claw 会自动读取这些配置。安装 Code Claw → 扫码绑定 → 直接开始用。
